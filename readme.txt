README

para aviar el juego en ambiente unix es suficiente ejecutar el script ./dipolo
en ambiente windows o mac se dobrá cargar dipolo.lisp en el interprete clisp: (load 'dipolo.lisp)

para juegar seguir las istruciónes en la pantalla

NB: cuando se pregunta de elijir la torre da muever es necesario ponerla en la forma (x y n) donde x es el numero de columna, y es la fila y n es el numero de piezas que hay en la torre.
